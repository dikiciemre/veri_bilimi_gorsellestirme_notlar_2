# veri_bilimi_gorsellestirme_notlar_2
İnternetten aldığım veri görselleştirme ve veri analizi eğitimleri sonucunda aldığım notları sizlerle paylaşmak istiyorum.
Bu notlar içerisinde Pandas, Numpy, Matplotlib,Seaborn kütüphanelerinden kapsamlı örnekler göreceksiniz.

Sizlere kısaca bu kütüphaneleri anlatmak istiyorum.
